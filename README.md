# GHOST_HOOK essentials

Runtime files used by the Project Delta scripts.

- `pinreta.lua` - main Pinreta script, configured to load dependencies from this repo.
- `ghost.lua` - Ghost script copy, configured to load dependencies from this repo.
- `test lib` / `library_main.lua` - Ghost-compatible UI wrapper.
- `mclib` - MacLib UI source.
- `radar` - radar runtime loaded when radar is enabled.
- `assets.json` plus `grad*.png`, `library_save.lua`, `library_theme.lua`, and `chat_spam.lua` - cached support files downloaded by the script.
- `Keybinds`, `Bal`, and `Key` - legacy Ghost dependencies.
